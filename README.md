# devnazgul — Portfolio & Links

This repository hosts a small static portfolio / link hub deployed to GitHub Pages.

How to run locally

- Open `index.html` in your browser, or run a local static server:
  - Python 3: `python -m http.server`
  - Node (serve): `npx serve .`

How to publish (GitHub Pages)

1. Go to Settings → Pages for this repository: https://github.com/devnazgul/profile/settings/pages
2. Under "Source" select the default branch (main) and the root folder (/), then Save.
3. Wait a minute or two; the site should be available at: https://devnazgul.github.io/profile

Files included

- `index.html` — the single-page site (already pushed).
- `styles.css` — styles for the site (already pushed).

Customizations

- Edit `index.html` to replace placeholders (name, avatar, links, projects).
- Add `resume.pdf` to the repo root to enable the Resume link.
- To publish at your root username site (https://devnazgul.github.io) rename the repository to `devnazgul.github.io` or push the same files into that repo.
